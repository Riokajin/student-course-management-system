## Project Overview
This project is a simple Student Course Management System built using:
- Object-Oriented Programming (classes, objects, encapsulation)
- Input validation
- Exception handling
- Java collections (ArrayList, HashMap, etc.)
 
The system will allow users to manage students, courses, and enrollments based on the assignment requirements.
